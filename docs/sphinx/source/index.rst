Hyper3 Documentation
=====================

Hyper3 is a self-evolving hypergraph knowledge graph library for Python.
It provides directed n-ary hyperedges, rule-based multiway inference,
Born-rule belief distributions, Bayesian updating, spreading activation
retrieval, temporal reasoning, and continuous structural self-evolution.
